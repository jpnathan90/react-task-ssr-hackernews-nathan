# Padmanathan REACT SSR HackerNews Task

## Prerequisites

```
node (version > 10)
npm

```

# Run development project locally

```
git clone https://github.com/jpnathan90/react-task-ssr-hackernews-nathan.git
npm install
npm run dev
open http://localhost:3000 to view project on the browser
```
